# **App Name**: AdSight

## Core Features:

- Generative Ad Auditor: Uses a vision model tool to interpret messaging clarity and identify target audience signals from uploaded image creatives.
- Integrated Drag-and-Drop Port: A high-performance upload zone supporting JPG, PNG, and marketing screenshots.
- Process Logic HUD: An interactive loading sequence with rotating progress updates evaluating visual hierarchy and call-to-action effectiveness.
- Insight Insights Matrix: A premium dashboard-style results screen categorizing strengths, opportunities, and overall creative assessment.
- Cloud Snapshot Sync: Secure storage utilizing a Firestore database to preserve advertisement analysis for rapid retrieval without local caching.
- Asset Action Suite: Utility features allowing users to copy textual analysis, export full PDF reports, or share insight links instantly.
- Dynamic Creative Overlay: Provides a side-by-side view of the original creative with AI annotations on visual flow and heat points.

## Style Guidelines:

- Primary color: Electric Indigo (#6366F1) for a sophisticated tech-forward identity.
- Background color: Deep Midnight Navy (#08080F) with integrated radial blurs of Indigo and Purple.
- Accent color: Vibrant Violet (#A855F7) used for progress glows and distinct marketing insight highlights.
- Headline font: 'Poppins' for a precise, avant-garde feel; Body font: 'Inter' for clean, Machined readability across dense analysis blocks.
- Monolinear icons with a neon stroke weight, reflecting modern premium SaaS standards.
- A spacious, immersive layout using frosted glass card effects and expansive negative space.
- Atmospheric blue and purple pulsing glows that simulate a heartbeat during active AI processing.
